Introduction

This file cleans the raw data by applying a simply dictionary of the main categories we analyse in “Gender Representation in Large Language Models: A Cross-Linguistic and Cross-Model Analysis”

Main

Extracting just the gender answers

results_parsed <- results |> 
  mutate(res = map2(
    .x = answer, 
    .y = lang,
    .f = function(.x, .y) apply_codebook(.x, codebook = gender_patterns[[.y]]),
    .progress = interactive()
  )) |> 
  unnest_wider(res)
