Introduction

This file does all the prompting for the article “Gender Representation in Large Language Models: A Cross-Linguistic and Cross-Model Analysis”

Packages

We use Ollama with the R package rollama for the prompting. To make the prompting more efficient and make sure errors can be easily recovered from, I use a custom version of httr2::req_perform_parallel(), which rollama uses under the hood.

Data

We construct a grid of the different prompts (one for each language), the selected nouns, and a range of different models.

results <- readRDS("data/results_parsed.rds") |> 
  select(prompt_id, noun_id, Noun, Valence, lang, noun, model, seed, temperature, answer)
# dictionary for gender extraction

Main

Extracting just the gender answers

results_parsed <- results |> 
  mutate(res = map2(
    .x = answer, 
    .y = lang,
    .f = function(.x, .y) apply_codebook(.x, codebook = gender_patterns[[.y]]),
    .progress = interactive()
  )) |> 
  unnest_wider(res)

Result validation

results_parsed |> 
  unnest_longer(category) |> 
  count(category, lang, sort = TRUE) |> 
  DT::datatable()

wrap up

dir.create("data", showWarnings = FALSE)
saveRDS(results_parsed, "data/results_final.rds")
results_parsed |>
  rowwise() |> 
  mutate(
    hit = toString(hit),
    category = paste(category, collapse = "_")
  ) |> 
  rio::export("data/results_final.csv")
